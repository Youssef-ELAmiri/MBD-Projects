# MBD Projects - Model-Based Design Portfolio

Bienvenue dans mon dépôt dédié aux projets développés selon la méthodologie **Model-Based Design (MBD)**. Ce repository illustre l'application du cycle en V dans la conception de systèmes embarqués complexes, allant de la spécification à la génération de code pour cibles matérielles.

## 🚀 Vision du Repository
L'ingénierie moderne exige de valider les concepts virtuellement avant toute implémentation physique. À travers ces projets, je démontre ma maîtrise de la chaîne d'outils MATLAB/Simulink pour :
- La modélisation de systèmes dynamiques.
- La conception de logiques de contrôle décisionnelles.
- La validation rigoureuse (MIL, SIL, PIL).
- Le déploiement sur microcontrôleurs (Arduino, ESP32, Raspberry Pi).
